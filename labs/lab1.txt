
### Задание 1: Создание нового репозитория и добавление файла

1. **Создайте новый репозиторий на GitHub.**
    - Назовите репозиторий `my-first-repo`.
    - Добавьте описание: "Мой первый репозиторий на GitHub".

2. **Клонируйте репозиторий на ваш локальный компьютер.**
    ```sh
    git clone https://github.com/<ваше_имя_пользователя>/my-first-repo.git
    ```

3. **Создайте новый файл в клонированной папке.**
    - Назовите файл `README.md`.
    - Вставьте следующий текст в файл:
      ```markdown
      # Мой первый репозиторий

      Это мой первый опыт работы с GitHub!
      ```

4. **Добавьте и закоммитьте файл.**
    ```sh
    git add README.md
    git commit -m "Добавлен файл README.md"
    ```

5. **Запушьте изменения на GitHub.**
    ```sh
    git push origin main
